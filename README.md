_**Train Reservation & Management System**_

ProjectSE – Software Engineering Project

A complete web-based system for managing train schedules, passenger profiles, ticket bookings, reports, and user authentication.
Built using HTML, CSS, JavaScript, and LocalStorage.


<br>
<br>

**Overview**  

      This project provides a full reservation workflow, including:

Schedule management

Passenger registration

Ticket booking

Revenue reports

User login & role-based access

Dashboard analytics

    The system is fully client-side and requires no backend server.

<br>
<br>

**Project Structure**

ProjectSE/

│── Booking/

<br>

* Authentication & User Roles
The system includes predefined users:

      | Username | Password | Role 

      | admin | admin123 | administrator 

      | agent | agent123 | reservation_agent 

      | manager | manager123 | manager 


<br>


* Booking Module

Features:

Seat availability check

Ticket booking

Automatic seat decrement

Booking history

Transactional rollback on failure

<br>


* Passenger Management Module
 
Features:

Add new passenger

Edit passenger profile

Delete passenger

Search passengers

Form validation

Duplicate passport detection

<br>


* Schedule Management Module
  
Features:

Add schedule

Edit schedule

Delete schedule

Display schedules in a table

Save schedules in LocalStorage

<br>


* Reports Module

Features:

Revenue calculation per train

Booking summaries

Data analysis
